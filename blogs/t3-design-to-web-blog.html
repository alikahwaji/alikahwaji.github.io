Mobile first design is an approach outlined in 2009 by Luke Wroblewski. Put simply, mobile first is an approach to responsive design: design for smaller screens first, then add more features and content for bigger and bigger screens.This design approach is also known as "progressive enhancement." Mobile first design is important for these reasons : 1-Mobile first design force you to focus on core content and functionality. 2-Screen sizes abound in the mobile web. 3-New capabilities with the mobile web. 4-Progressive enhancement.
