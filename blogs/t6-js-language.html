Javascript Language

In the JavaScript language describe all the use cases for each of the follwing:

  1. Parenthesis() : in JavaScript are used for function calls, to surround conditional statements, or for grouping to enforce Order of Operations.
  2. Braces {}: are used during the declaration of Object Literals, or to enclose blocks of code (function definitions, conditional blocks, loops, etc).
  3. Brackets[ ]: they are used with arrays. Arrays are a more advanced form of variables and will be covered in a separate tutorial.
  4. single quotes ' ':single quotes can be used for strings.
  5. double quotes " ": Double quotes eliminate the need to escape apostrophes (as in contraptions). Consider the string: "I'm going to the mall", vs. the otherwise escaped version: 'I\'m going to the mall'.
